## 📘 About the Project

The **Kindness Cabinet – Donation & Resale App** is a **real-world mobile application** developed as a  
**Final Year IT Project**.

It enables users to **donate, resell, or share unused items** such as clothes, books, gadgets, toys, furniture, and household goods through a simple and secure Android application.

The project focuses on **sustainability, social responsibility, and digital inclusion** by connecting donors, buyers, and underprivileged communities.
---

## 🌍 Project Overview

🌱 This platform allows users to:
- Donate or resell items digitally  
- Browse products by category and location  
- Communicate securely via in-app chat  
- Track donations, orders, and resale history  
- Promote eco-friendly reuse and reduce waste  

💡 Built to demonstrate **real-life Android + PHP + MySQL application development** with social impact.

---

## 🔧 Key Features

| Feature | Description |
|------|-------------|
| 👤 Secure Authentication | Google Sign-In, OTP login, session management |
| 📦 Item Listing & Management | Upload items with images, category & donation/resale option |
| 🗂 18+ Categories | Clothes, Books, Gadgets, Furniture, Toys, Kitchenware, Shoes & more |
| 🔎 Smart Search & Filters | Price, location, donation-only, condition |
| 📝 Wishlist & Favorites | Save items for later |
| 💬 In-App Chat System | Secure donor–receiver communication |
| 🔔 Real-Time Notifications | Messages, offers & updates |
| 🛒 Payment & Donation Flow | Online payment & digital donation tracking |
| 📊 Order History & Analytics | Track resales, donations & contributions |
| 🛡 Admin Dashboard | User management, approvals & fraud detection |
| 🌐 Location-Based Services | Google Maps API integration |
| 🔒 Security Features | OTP, encrypted login, SQL injection prevention |
